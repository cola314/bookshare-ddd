# 이벤트 스토밍

## 1단계: 도메인 이벤트 (🟧 오렌지)

- 리뷰를 작성했다
- 리뷰를 수정했다
- 리뷰를 삭제했다
- 로그인했다
- 로그아웃했다
- 리뷰에 좋아요를 눌렀다
- 리뷰에 좋아요를 취소했다
- 리뷰에 댓글을 남겼다
- 댓글을 삭제했다
- 관리자가 공지사항을 등록했다
- 관리자가 공지사항을 수정했다
- 관리자가 공지사항을 제거했다
- 사용자가 문의를 등록했다
- 사용자 이름을 변경했다

## 2단계: 커맨드 (🟦 파랑)

- 리뷰 작성하기 → 리뷰를 작성했다
- 리뷰 수정하기 → 리뷰를 수정했다
- 리뷰 삭제하기 → 리뷰를 삭제했다
- 로그인하기 → 로그인했다
- 로그아웃하기 → 로그아웃했다
- 좋아요 누르기 → 리뷰에 좋아요를 눌렀다
- 좋아요 취소하기 → 리뷰에 좋아요를 취소했다
- 댓글 작성하기 → 리뷰에 댓글을 남겼다
- 댓글 삭제하기 → 댓글을 삭제했다
- 공지사항 등록하기 → 관리자가 공지사항을 등록했다
- 공지사항 수정하기 → 관리자가 공지사항을 수정했다
- 공지사항 삭제하기 → 관리자가 공지사항을 제거했다
- 문의 등록하기 → 사용자가 문의를 등록했다
- 이름 변경하기 → 사용자 이름을 변경했다

## 3단계: 액터 (🟨 노랑)

- 비회원 (게스트)
- 회원 (로그인한 사용자)
- 관리자

## 4단계: 애그리거트 (🟨 연노랑)

- 리뷰 (Review)
- 좋아요 (Like)
- 댓글 (Comment)
- 회원 (Member)
- 공지사항 (Notice)
- 문의 (Inquiry)

- 문의

## 이벤트 스토밍 배치도

> 🟨 액터 → 🟦 커맨드 → 🟨 애그리거트 → 🟧 이벤트

### 리뷰 컨텍스트 (Review Context)

```
🟨회원 → 🟦리뷰 작성하기 → 🟨[리뷰] → 🟧리뷰를 작성했다
🟨회원 → 🟦리뷰 수정하기 → 🟨[리뷰] → 🟧리뷰를 수정했다
🟨회원 → 🟦리뷰 삭제하기 → 🟨[리뷰] → 🟧리뷰를 삭제했다
🟨회원 → 🟦좋아요 누르기 → 🟨[좋아요] → 🟧리뷰에 좋아요를 눌렀다
🟨회원 → 🟦좋아요 취소하기 → 🟨[좋아요] → 🟧리뷰에 좋아요를 취소했다
🟨회원 → 🟦댓글 작성하기 → 🟨[댓글] → 🟧리뷰에 댓글을 남겼다
🟨회원 → 🟦댓글 삭제하기 → 🟨[댓글] → 🟧댓글을 삭제했다
```

### 회원 컨텍스트 (Member Context)

```
🟨회원 → 🟦로그인하기 → 🟨[회원] → 🟧로그인했다
🟨회원 → 🟦로그아웃하기 → 🟨[회원] → 🟧로그아웃했다
🟨회원 → 🟦이름 변경하기 → 🟨[회원] → 🟧사용자 이름을 변경했다
```

### 고객지원 컨텍스트 (Support Context)

```
🟨관리자 → 🟦공지사항 등록하기 → 🟨[공지사항] → 🟧관리자가 공지사항을 등록했다
🟨관리자 → 🟦공지사항 수정하기 → 🟨[공지사항] → 🟧관리자가 공지사항을 수정했다
🟨관리자 → 🟦공지사항 삭제하기 → 🟨[공지사항] → 🟧관리자가 공지사항을 제거했다
🟨비회원/회원 → 🟦문의 등록하기 → 🟨[문의] → 🟧사용자가 문의를 등록했다
```
