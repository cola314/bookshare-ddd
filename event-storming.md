# 이벤트 스토밍

## 1단계: 도메인 이벤트 (🟧 오렌지)

- 리뷰를 작성했다
- 리뷰를 수정했다
- 리뷰를 삭제했다
- 로그인했다
- 로그아웃했다
- 리뷰에 좋아요를 눌렀다
- 리뷰에 좋아요를 취소했다
- 리뷰에 댓글을 남겼다
- 댓글을 삭제했다
- 관리자가 공지사항을 등록했다
- 관리자가 공지사항을 수정했다
- 관리자가 공지사항을 제거했다
- 사용자가 문의를 등록했다
- 사용자 이름을 변경했다

## 2단계: 커맨드 (🟦 파랑)

- 리뷰 작성하기 → 리뷰를 작성했다
- 리뷰 수정하기 → 리뷰를 수정했다
- 리뷰 삭제하기 → 리뷰를 삭제했다
- 로그인하기 → 로그인했다
- 로그아웃하기 → 로그아웃했다
- 좋아요 누르기 → 리뷰에 좋아요를 눌렀다
- 좋아요 취소하기 → 리뷰에 좋아요를 취소했다
- 댓글 작성하기 → 리뷰에 댓글을 남겼다
- 댓글 삭제하기 → 댓글을 삭제했다
- 공지사항 등록하기 → 관리자가 공지사항을 등록했다
- 공지사항 수정하기 → 관리자가 공지사항을 수정했다
- 공지사항 삭제하기 → 관리자가 공지사항을 제거했다
- 문의 등록하기 → 사용자가 문의를 등록했다
- 이름 변경하기 → 사용자 이름을 변경했다

## 3단계: 액터 (🟨 노랑)

- 비회원 (게스트)
- 회원 (로그인한 사용자)
- 관리자
