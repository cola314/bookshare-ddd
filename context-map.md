# 컨텍스트 맵 (Context Map)

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                  ┌──────────────────┐                       │
│                  │  회원 컨텍스트    │                       │
│                  │    (Upstream)    │                       │
│                  └────────┬─────────┘                       │
│                           │                                 │
│            ┌──────────────┼──────────────┐                  │
│            │              │              │                  │
│            ▼              │              ▼                  │
│  ┌──────────────────┐     │    ┌──────────────────┐         │
│  │  리뷰 컨텍스트    │     │    │ 고객지원 컨텍스트  │         │
│  │  (Downstream)   │     │    │  (Downstream)    │         │
│  │                 │     │    │                  │         │
│  │  - 작성자 정보   │     │    │  - 문의자 정보    │         │
│  └──────────────────┘     │    └──────────────────┘         │
│                           │                                 │
└─────────────────────────────────────────────────────────────┘
```

## 컨텍스트 간 관계

| Upstream | Downstream | 관계 유형 | 설명 |
|----------|------------|-----------|------|
| 회원 | 리뷰 | Customer/Supplier | 리뷰 작성자 정보 제공 |
| 회원 | 고객지원 | Customer/Supplier | 문의자 정보 제공 |

## 통합 방식

- **리뷰 컨텍스트**: 회원 ID만 참조, 필요 시 회원 컨텍스트에 조회
- **고객지원 컨텍스트**: 회원 ID만 참조 (비회원은 ID 없음)
